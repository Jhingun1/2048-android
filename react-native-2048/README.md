# React Native 2048

A React Native implementation of the 2048 game built with Expo and TypeScript.

## Features

- Swipe gestures for tile movement
- Score and best score tracking
- New game and undo controls
- Win and game-over overlays

## Run locally

```bash
cd react-native-2048
npm install
npm run start
```

Then press:

- `a` for Android emulator/device
- `i` for iOS simulator (macOS only)
- `w` for web

## Type checking

```bash
npm run typecheck
```
